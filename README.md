- 🎴 **24张精心设计的卡牌** - 涵盖自信、梦想、治愈、勇气、智慧等多个主题
- 🎲 **随机抽卡功能** - 每次都能获得独特的启示和鼓励
- 💫 **精美的卡片设计** - 炫彩渐变背景和流畅的3D翻转动画
- 📱 **响应式布局** - 完美支持桌面、平板和手机设备
- 🌙 **沉浸式体验** - 简洁优雅的界面，宁静祥和的氛围

## 🚀 如何使用

1. **抽卡** - 点击"抽卡"按钮，随机获得一张卡牌
2. **查看解读** - 卡牌会自动翻转，显示卡牌名称、关键词和详细解读
3. **再次翻转** - 点击卡牌可以在正面和反面之间切换
4. **重置** - 点击"重置"按钮回到初始状态
5. **查看全部** - 点击"查看全部卡牌"浏览所有24张卡牌
